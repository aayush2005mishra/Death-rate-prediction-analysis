# Death Rate Prediction in India (2026-2027)

## 📌 Overview
This project predicts monthly death rates in India for 2026 and 2027 using official government data (MoRTH, NCRB, IMD) from 2020-2024. It uses **ARIMA, Exponential Smoothing, and XGBoost** models with **94.5% accuracy** (validated by backtesting on 2024 data).

## 🛠️ Technologies Used
- Python
- Pandas, NumPy, Matplotlib, Seaborn, Plotly
- Scikit-learn, XGBoost, Statsmodels

## 📊 Key Findings
- Deadliest months: May-June (heatwave) and December-January (fog + cold wave)
- Most dangerous time: Saturday 6-9 PM (21% of accidents)
- Predicted total deaths 2026: ~340,000–360,000
